[Vim] 最低限必要な設定コマンド

Jan 21, 2021

欲しい・あると便利な設定ではなく、まっさらなVim環境nに遭遇したときにさっと手打ちで設定すべき最低限必要な設定コマンドをまとめます。

設定コマンド

行番号表示

set nu

シンタックスハイライト

syntax on

スマートインデント

set smartindent

タブ文字を表示

set listchars=tab:>-

これは丸暗記で。

設定ファイルに書く

ログインしたアカウントのホームディレクトリに下記ファイルを作り、その中に上記設定コマンドを書く。

.vimrc
